✏️ 专题二 · 分型·笔·线段 第62、65、67、69、71、77、78课

1. 分型

📐 定义 · 顶分型

相邻三根K线，中间K线的最高点是三者中最高的，最低点也是三者中最高的。
顶 形态：↑ ↑ ↑ 中间最高

📐 定义 · 底分型

相邻三根K线，中间K线的最低点是三者中最低的，最高点也是三者中最低的。
底 形态：↓ ↓ ↓ 中间最低

🧠 分型心理意义

顶分型：多空交战，空方胜出（第三K线跌破第一K线最低点 → 强势顶分）
底分型：空多交战，多方胜出（第三K线升破第一K线最高点 → 强势底分）
中继分型：分型后反向走势没能延续，属于虚假信号

2. 笔

✏️ 定义 · 笔

两个相邻的顶和底之间构成一笔。
向上笔：底分型 + K线序列 + 顶分型
向下笔：顶分型 + K线序列 + 底分型
⚡ 要求：顶和底之间至少有一根独立的K线（不含分型本身的K线）

📏 笔的严格划分规则
  • 顶分型的最高点和底分型的最低点之间，必须有至少一根独立K线
  • 同一笔中，顶分型不能共用K线
  • 必须先处理K线的包含关系（相邻K线高低点完全覆盖）
  • 新笔定义（宽松版）：只要顶底之间满足5根以上K线即可

3. 线段

📏 定义 · 线段

线段由连续的三笔或三笔以上构成，且前三笔必须有重叠部分。线段的终结：被反向的线段所破坏。

📏 线段划分标准（第67课·核心）

特征序列法：

  • 将线段中的笔，按方向分成特征序列
  • 向上线段 → 向下笔为特征序列元素；向下线段 → 向上笔为特征序列元素
  • 标准特征序列的顶/底分型出现 → 线段被破坏
💡 线段破坏的两种方式

第一种情况：特征序列的分型第一元素和第二元素之间没有缺口 → 直接破坏
第二种情况：特征序列的分型第一元素和第二元素之间有缺口 → 需要第三元素确认

🎯 专题三 · 中枢·走势类型·买卖点 核心篇

1. 走势中枢（核心基石）

📐 定义 · 缠中说禅走势中枢

某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

📜 缠中说禅走势中枢定理一

任何级别的走势中，中枢由三个连续次级别走势类型的重叠部分构成。
中枢区间 = 三个次级别走势类型的重叠区间。

📜 缠中说禅走势中枢定理二

中枢的形成：连续三个次级别走势类型有重叠区间
中枢的延伸：中枢形成后，次级别走势类型继续围绕中枢区间运动
中枢的新生：产生新的中枢，与旧中枢不重叠 → 趋势
中枢的扩张：形成新的中枢，与旧中枢重叠 → 更大级别中枢

📏 中枢的级别扩展

中枢震荡延伸超过9段（即9个次级别走势类型重叠）→ 中枢级别升级
例：5分钟中枢震荡延伸9段以上 → 升级为30分钟级别中枢

2. 走势类型

📐 定义 · 走势类型

趋势：包含至少两个以上依次同向的缠中说禅走势中枢
  └ 上涨 中枢依次向上移动
  └ 下跌 中枢依次向下移动
盘整：包含唯一一个缠中说禅走势中枢

📋 走势类型标准结构

上涨标准式：a + A + b + B + c
下跌标准式：a + A + b + B + c
盘整：a + A + b
其中：a、b、c为次级别走势类型（连接段），A、B为中枢

📜 缠中说禅技术分析基本原理一

走势终完美：任何级别的任何走势类型终要完成。即：任何走势（上涨、下跌、盘整）都必然结束 → 转入另一种走势类型。

📜 缠中说禅技术分析基本原理二

任何级别的任何完成的走势类型，必然包含一个缠中说禅走势中枢。

3. 三类买卖点

🎯 第一类买点 买1
某级别下跌趋势中，最后一个中枢的背驰点。本质：趋势力量衰竭 → 转折。
🔔 MACD：黄白线回拉0轴后，背驰段绿柱面积缩小
🎯 第一类卖点 卖1
某级别上涨趋势中，最后一个中枢的背驰点。本质：上涨力量衰竭 → 转折。
🔔 MACD：黄白线回拉0轴后，背驰段红柱面积缩小
🎯 第二类买点 买2
第一类买点之后的次级别回抽不破前低。本质：趋势背驰后的回踩确认。
🔔 常与第一类买点形成"双底"形态
🎯 第二类卖点 卖2
第一类卖点之后的次级别回抽不过前高。本质：趋势背驰后的反弹确认。
🔔 常形成"M头"右肩
🎯 第三类买点 买3
离开中枢的次级别走势，其回抽不进入中枢区间。本质：中枢突破后的回踩确认。
🔔 最强的买点类型，适合快速拉升段
🎯 第三类卖点 卖3
离开中枢的次级别走势，其回抽不回到中枢区间。本质：中枢破位后的反抽确认。
🔔 必须立即离场的信号
📜 缠中说禅买卖点定律

所有买卖点都可以归结到第一类买卖点，只是级别不同。第二类买卖点 = 次级别的第一类买卖点。第三类买卖点 = 次级别离开 + 次级别回抽不进入中枢。

📜 缠中说禅买卖点分析的完备性

在缠论中，只有三类买卖点，没有第四类。任何操作都可以明确归入这三类中的某一类。

⚡ 专题四 · 背驰篇 第15、24、25、27、29、37、43、44课

📐 定义 · 背驰

趋势背驰：某级别趋势（至少两个中枢）的最后一段走势力度弱于前一段同向走势。
盘整背驰：盘整走势（一个中枢）的进入段和离开段的力度比较。

📏 背驰判断方法（MACD辅助）
  1. 确认级别：必须在同一级别图上比较
  2. 两段比较：比较中枢前后两段同向次级别走势
  3. MACD判断：后段对应的MACD红绿柱面积明显小于前段
  4. 黄白线：后段黄白线高度低于前段（回拉0轴后再次离开）
  5. 确认：后段内部出现次级别背驰（区间套）
📜 缠中说禅背驰-买卖点定理

任一背驰都必然制造某级别的买卖点，任一级别的买卖点都必然源自某级别走势的背驰。
背驰是买卖点的充分必要条件。

📜 缠中说禅背驰-转折定理

某级别趋势的背驰将导致：
① 该趋势最后一个中枢的级别扩展（最小力度转折）
② 该级别更大级别的盘整
③ 该级别以上级别的反趋势（最强力度转折）

📜 缠中说禅小背驰-大转折定理

小级别顶背驰引发大级别向下的必要条件：该级别走势的最后一个次级别中枢出现第三类卖点。
小级别底背驰引发大级别向上的必要条件：该级别走势的最后一个次级别中枢出现第三类买点。

🔗 专题五 · 级别·递归·同级别分解 第35、36、38、39、40课

📐 级别概念

级别是缠论的核心维度。不同级别如同不同倍数的显微镜：
1分钟图 → 精密微观 | 5分钟图 → 中级视野 | 30分钟图 → 战术级别 | 日线/周线 → 战略方向

📏 区间套定位法

在大级别图上找到背驰段，然后逐级缩小到小级别图寻找精确背驰点。例：日线背驰段 → 30分钟图 → 5分钟图 → 1分钟图 → 最后精确买点。

🧮 同级别分解

原则：按某固定级别将所有走势分解为同级别走势类型的连接。
优点：避免多级别分析的复杂性，适用于固定级别操作者。
赋格（第40课）：可同时按多个级别进行同级别分解，形成多声部操作。

📜 缠中说禅走势分解定理一

任何级别的任何走势，都可以分解成同级别"盘整"、"下跌"、"上涨"三种走势类型的连接。

📜 缠中说禅走势分解定理二

任何级别的任何走势类型，都至少由三段以上次级别走势类型构成。

级别递归关系

  • 笔 → 最低级别构件 | 线段 → 三笔构成（最低级别走势类型）
  • 1分钟中枢 → 三个1分钟线段重叠 | 1分钟走势 → 含至少一个1分钟中枢
  • 5分钟中枢 → 三个1分钟走势重叠 | 以此类推……递归定义

🌀 专题六 · 中阴阶段 第89、90课

📐 定义 · 中阴阶段

走势类型完成之后、新的走势类型尚未确认之前的过渡阶段。表现为：原走势中枢的震荡延伸，方向未定。

📏 中阴阶段结束的辅助判断

MACD辅助法：①中阴阶段对应MACD黄白线回拉0轴 ②黄白线在0轴附近缠绕 ③最终出现强力突破（第三类买卖点）→ 中阴结束。
布林线辅助：中轨走平的时间为中阴阶段长度参考。

⚠️ 中阴阶段操作纪律
  • 中阴阶段不预测方向，只跟随
  • 按中枢震荡操作：下沿买、上沿卖
  • 出现第三类买卖点才确认方向
  • 级别越大的中阴，震荡周期越长

📊 专题七 · 走势结构的两重表里关系 第91-93、99课

📐 两重表里关系

第一重：走势类型（表） vs 中枢结构（里）
第二重：分型（表） vs 走势类型（里）
如同中医的表里辨证，通过表（可见形态）推断里（内在能量）。

📏 分型-走势类型表里关系

(−1,1) 向下笔延伸中 → 对应下跌走势的可能
(1,1) 向上笔延伸中 → 对应上涨走势的可能
(1,0) 出现顶分型（向上笔可能结束）→ 警惕卖点
(−1,0) 出现底分型（向下笔可能结束）→ 关注买点

💡 操作意义

当机立断（第94课）：在(1,0)出现时，如果大级别也处于背驰段，就必须果断卖出。四个状态周期循环：(−1,1) → (−1,0) → (1,1) → (1,0) → (−1,1) → ...

⚔️ 专题八 · 实战图解篇 第54-61课

📜 缠中说禅第一利润最大定理

对于任何固定交易品种，在确定的操作级别下，按买点买、卖点卖的操作模式利润率最大。

📜 缠中说禅第二利润最大定理

对不同交易品种进行轮动操作，在多个品种间切换，利用不同品种的买卖点时间差获取更大利润。

短线操作法（第79课·三买买卖法）

  1. 选定日线第三买点的股票池
  2. 用30分钟图找精确买点
  3. 买入后按5分钟级别的卖点出局
  4. 如果日线第三买点后走强 → 转为30分钟级别操作

操作格言

  • 买点买，卖点卖 —— 市场的节奏，只有一个
  • 没有节奏，只有死（第41课）
  • 不带套的操作不是好操作（第13课）—— 严格止损
  • 走势终完美 —— 永远不追高，等待下一个买点

💰 专题九 · 资金管理 第22、31课

📐 资金管理原则
  • 分仓操作：永不单票满仓
  • 级别匹配：大资金用大级别，小资金用小级别
  • 分批出入：买点区域分批建仓，卖点区域分批减仓
  • 利润保护：盈利后逐步收回本金，用利润博弈

本ID理论站在现代物理角度构建能量动力学结构。一切几何结构说了算：走势的几何结构决定能量动力形态。结构是骨架，能量是血肉。

📈 专题十三 · 均线系统与实战技巧 第11-14、103、106-108课

📐 吻系统

飞吻：短期均线略略接触长期均线即继续原方向
唇吻：短期均线靠近长期均线但不交叉
湿吻：短期均线反复缠绕长期均线（最危险也最可能转折）
女上位 短期均线在长期均线之上 → 多头市场
男上位 短期均线在长期均线之下 → 空头市场

⚠️ 缠中说禅防狼术（第103课）

MACD在0轴之下时，绝不买入。MACD0轴以下 = 空头主导，任何反弹都是卖出机会而非买入机会。
