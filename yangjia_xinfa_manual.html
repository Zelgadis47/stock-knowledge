一、心法总纲 · 六条核心

养家心法的精髓提炼——每一条背后都是对市场心理的深刻洞察。

心法一
高手买入龙头，超级高手卖出龙头
当题材发酵、龙头股成为"大众情人"时，风险已在悄然累积。
真正的能力不是买在起涨点，而是在最狂热时从容离场。
龙头战法逃顶逆向思维
心法二
别人贪婪时我更贪婪，别人恐慌时我更恐慌
赚了钱的人会继续追涨，亏了钱的人会继续杀跌。情绪本身具有自我强化和延续的惯性。
在赚钱效应弥漫时全力做多，在亏钱效应扩散时果断空仓——这不是反人性，而是顺应情绪周期的客观规律。
情绪周期顺势仓位管理
心法三
敢于大盘低位空仓，敢于大盘高位满仓
大多数人依据历史点位判断高低。大盘从5000跌到3000，算"低位"——但可能跌到2000。
正确做法：赚钱效应强就参与，亏钱效应强就回避，与点位无关。
逆人性不预测只跟随
心法四
永不止损，永不止盈
真正的操作依据是风险收益比的动态评估，而非机械的止损止盈线。
不看好了就卖出，有更好的标的就换仓，心中没有成本，自然没有止损。
心理博弈成本无关主动操作
心法五
得散户心者得天下，人气所向，牛股所在
题材的级别决定想象空间，赚钱效应决定资金参与热情。
一个没有群众基础的板块走不远，一个没有人气的股票涨不高。
题材级别人气赚钱效应
心法六
心中有顶底，操作自随心
预判不是目的，应变才是根本。不用事先设定顶和底，根据市场情绪的演变动态调整操作策略。
应变动态判断不预判
二、情绪周期论 · 市场心理的动态演变

养家心法的核心框架——一切操作的基础都是对市场情绪的揣摩。

核心公式
赚钱效应 → 追涨资金赚钱 → 自信心膨胀 → 继续追涨 → 行情推向高潮 → 赚钱效应告一段落

亏钱效应 → 追涨资金亏钱 → 自我惩罚 → 操作趋于谨慎 → 活跃度下降 → 崩溃逐步酝酿
自我强化正反馈负反馈
崩溃与反转 — 情绪周期的关键拐点
崩溃前兆：抄底的人也在亏钱 → 抄底行为减少 → 稍有卖压便无抵抗式下跌 → 加速崩溃
反转条件：大面积跌停（几十到上百家）+ 部分品种跌幅达 20-30%+ → 超跌品种企稳 + 新热点萌芽 → 情绪逆转
⚠️ 关键认知："已经这么低了还能跌到哪里去"是危险的。正是因为大多数人有这种想法，才导致崩溃发生时抛压更集中。
崩溃盘 情绪逆转 左侧交易 恐慌释放
三、热点分类 · 四层操作体系
类型特征操作策略典型案例
主流热点 涨幅大、跨度时间长，国家大级别战略驱动 反复做。第一波赶上了就做足，没赶上等回落做第二波 页岩气、文化传媒、自贸区、稀土
支流热点 各支流循序轮动，围绕主流逻辑逐步挖掘 在主流不倒的前提下，以主流为中心挖掘扩散 水泥跟随保障房、水利跟随基建
次主流 涨幅和时间跨度少于主流，但仍具可操作性 围绕龙头操作，周期短于主流 券商、环保、舟山概念
非主流 时间短，靠技术面或消息面短期爆发 快进快出，隔日超短；爆发初期可参与，后市不追高 摘帽行情、一夜情题材
题材级别判断法则
国家大战略级别 > 行业政策 > 突发事件 > 个股消息

判断标准：是否符合国家核心利益？政策支持力度是否持续反复出现（如新闻联播高频提及）？
例：文化传媒 — 政策由投资转向消费的战略拐点，生逢其时，反复活跃数月
四、市场四阶段 · 对应操作策略
📈 强势阶段
操作：追涨龙头 — 赚钱效应下的资金不断进入，强的标的被后来资金选中的概率更大，所以选强。
买股逻辑：因为赚钱效应 → 资金不断涌入 → 强者恒强
追涨龙头强者恒强
📉 弱势初期
操作：低吸止损盘 — 强势品种还有主升能力的做回调后第二波。
关键判断：超跌品种稳住 → 强势股有机会；超跌品种继续创新低 → 强势股补跌在即
低吸强势股回调
💀 弱势中期
操作：低吸崩溃盘 — 超跌的板块在连续下跌后再次出现集体性大幅杀跌时出手。
核心：用轻仓试错保持盘感，等待真正的恐慌释放重仓出击
崩溃盘恐慌轻仓
🔥 弱势末期
操作：追涨强势 — 恐慌释放殆尽，由弱转强阶段是暴利机会。
提前埋伏回调充分、有题材有想象力的品种，等待右侧确认
转强暴利右侧
📊 数据统计印证：养家自述收益 95% 以上来自右侧交易（强势阶段与弱势末期）。
左侧模式（弱势初、中期）更多是为了保持盘感——输赢不重要，重要的是感受市场情绪。
五、实战技法精要
技法一 · 低吸崩溃盘
条件：大盘连续下跌一段后 → 再出现大阴线 → 指数远离 5 日线
配合：板块在连续下跌后再次出现集体性大幅杀跌
双重共振时，即为超跌出击时刻。
左侧交易恐慌大阴线
技法二 · 龙头套利链条
套利逻辑：龙头超预期上涨 → 板块内其他低位品种产生补涨空间
操作方法：龙头连板时，挖掘同一板块中低位、小盘、低价且未大涨的品种
经典案例：山东墨龙 → 宝莫股份；深天马 → 华东科技
补涨比价效应联动
技法三 · 做T 心法
不是为了做T而做T。T = 机会大时加仓 + 风险大时减仓，恰好发生在同一天同一只股票上，就成了"做T"。
本质是动态的买卖操作，不是机械的高抛低吸。
滚动操作动态非机械
技法四 · 仓位管理
赢面 vs 仓位的对应关系：
赢面 < 60% → ❌ 观望
赢面 60-70% → 🔸 小仓出击
赢面 70-80% → 🔶 中仓出击
赢面 80-90% → 🔴 大仓出击
赢面 > 90% → 🟢 满仓（需同时满足：上涨空间至少 30-50% + 下跌空间控制在 3-5% 以内）
风险收益比凯利公式赢面
技法五 · 主流热点买入法
选股顺序：① 先判定是否主流热点 → ② 如是，不必等涨停，可随时先买入 → ③ 确认强度后加仓
如果不是主流热点：等强度够就排队龙头，大幅超预期就同时排，注意仓位控制
核心：买的那一刻，后市敢看 30-50% 以上空间才重仓
分步建仓确认加仓空间预期
技法六 · 持筹者心理博弈
持筹者问自己： - 同样持筹的人在想什么？"给个反弹我就走" → 那你最好尽快走 - "让我解套我就走" → 你在他们解套之前走
持币者问自己：如果下跌会买吗？如果反弹会追吗？
让自己始终处于主动的一方。
心理博弈主动先手
技法七 · 崩溃前识别
三大前兆信号：
① 领跌品种中出现近期强势股 → 打击追涨热情
② 90/120 天周期级别的品种（煤炭、水泥等）再度领跌 → 抄底资金被套
③ 抄底的人也开始亏钱 → 引发无抵抗式下跌
风险预警系统性风险回撤控制
六、经典问答精粹 · 核心智慧
Q：如何判断大盘要崩溃？
A：特征上有较大面积跌停（几十到上百家），幅度上不少数品种有20-30%以上跌幅。然后观察各阶段超跌品种以及新热点反应，等待情绪逆转的一刻。
更重要的：这不只是一个判断，更是一个动态的应对过程。当下先空仓或低仓位回避这种概率，会使自己操作上更为主动。
Q：如何避免"追在山顶、割在谷底"？
A：先判定是否主流热点。如是，不必等涨停可随时先买入，等确认强度后再行加仓。如果不是主流热点，等。强度够高就排队龙头。
Q：如何选到有大空间的热点？
A：大局观是一个综合，宏观政策面是权重最大的因素——这是爆发力牛股产生的最大根源。技术只是辅助作用。符合主流热点的新故事，往往是赚大钱的机会。
Q：如何克服"拿不住好股票"的毛病？
A：操作被情绪主导才会拿不住。判断买卖的唯一标准只有风险和收益的比较——买入机会，卖出风险，只做对的交易，胜负交给概率，多一份淡定，少一分胜负心。
Q：对初学者有什么建议？
A：纯K线技术的打板，统计胜率约在45%左右，简单所谓的追龙头模式，胜率也大抵如此，大致于澳门赌场。要有大局观，重仓买入的一刹那要有后市还有3到5个涨停空间的判断。如果你没有后市的把握，一个板也不要重仓买。
Q：如何控制回撤？
A：控制回撤最重要的就是回避系统性崩溃风险。绝大部分崩溃都有前兆，从赚钱和亏钱效应的演变过程可以进行推断。不过为此可能也会放弃很多机会。看看天气，如果觉得要下雨了，就早点回家，不要贪玩，天气好了再出来。
Q：什么情况下可以重仓出击？
A：一年通常4-5次重仓必杀，要满足：① 胜率要求90%以上 ② 上涨空间至少看30-50% ③ 下跌空间控制在3-5%以内。符合条件时，有一种"哪怕少买了100股都觉得遗憾"的感觉。
Q：如何让预期更贴近市场实际走势？
A：以史为鉴，可以知兴替。掌握以往热点轮动规律后，自然预期的准确性会提高。但更为重要的是——判断为末，应变为本。
七、龙飞虎仓位管理 · 补充精华
滚动分仓模式
风格：滚动、分仓。品种大都是选择热门股（资金活跃的品种）+ 中线活跃品种的反复参与。
一般收盘持有 3-6 成仓位。第2天如果手里的股票出现大幅拉升会抛出，视情况轻重仓位。
核心四步
① 出击时间：必须把握住大盘否极泰来的临界点——借大盘的势
② 认清热点：借板块的势，或借赚钱示范效应的势
③ 个股选择：即时图走势强于同类个股，有涨停潜力的优先
④ 四两拨千斤：做好前两点，自然能体会——启动板块和大盘见底的时间刚好重合共振时，此板块就是阶段性主流
板块炒作规律
主流热点三阶段：
① 龙头大幅上涨，板块内尚未跟风 → ② 媒体开始提及，短线客追涨，板块浮出水面 → ③ 人人皆知，鸡犬升天，主力出货

关键判断：板块内最冷门的股票也开始上涨 → 板块循环已到末期 → 主力大举出货之时

量能核心：量能是超短强势股中非常重要的观察点。看量价之间的关系，不同阶段、不一样的量能。量能分析可以充分解读主流资金想法。
八、养家语录 · 值得反复咀嚼的金句
「交易的本质是群体博弈——随时衡量场外的潜在买入者的钱的数量和买入倾向，与场内筹码的数量和卖出倾向。」
「买入倾向的推理，大多来自赚钱效应；卖出倾向的推理，大多来自亏钱效应。」
「掌握市场之心，胜利接踵而至；心被市场掌握，失败连绵不绝。」
「如果你能掌握市场的情绪，胜利将接踵而来；如果你的情绪被市场掌握，失败将挥之不去。」
「如果说股市有什么规律的话，这种情绪转变的过程就是规律，而且可以预期的一千年内，这种规律亦不会有大的改变。」
「买股之前问自己：如果下跌了怎么办？如果回答是继续买，那么可以买；如果回答是不知道，那么就不买。」
「我这个股票如果不是主流热点，就不买；如果是主流热点，就买，买了以后持有一段时间直到没有上涨动力为止。」
「把所有心思放在揣摩市场情绪上，做好比价效应的挖掘，热点会自己到你面前来。」
「我们要利用市场的情绪来赚钱，自然也要避免自己陷入情绪的漩涡。」
「一个股票，买的时候问自己：往上敢不敢看30-50%的空间？往下敢不敢越跌越买？如果回答是肯定的，那就买吧。」
