核心交易思想

#
Principle 01

🌤️ 市场氛围第一

"第一，市场氛围。这是重中之重，如果跌幅榜上躺着众多跌停或者大跌百分5以上的个股，意味着市场氛围很差，大环境不好，这时候要小心，不轻易出手，尽量等尾盘。"

Principle 02

💰 赚钱效应是本质

"瞬息万变中的核心不变才是市场的本质，四个字，赚钱效应。"
"我复盘主要是看昨天的涨停板今天的赚钱效应，哪一类涨停股次日有肉，哪一类股持续有肉。"

Principle 03

🧠 盘感 = 长期积累

"对市场2000多只个股基本面了如指掌，然后每天坚持复盘"、"大多数上市公司的官方网站我都看过"、"唯手熟尔，长期形成的记忆烙印。"

Principle 04

⚔️ 执行力大于预判

🎯 八分把握才出手

打板体系

#

打板三要素（按重要程度排序）

  1. 1

    🌤️ 市场氛围（第一优先级）

    整体大环境决定出手与否。跌幅榜上众多跌停或大跌5%以上的个股 → 市场氛围差 → 不轻易出手，尽量等尾盘。大盘走势、涨跌家数比、涨停/跌停数量是核心监测指标。

  2. 2

    🔥 个股人气与股性（第二优先级）

    近期经常涨停的股票说明股性好、有资金关注。次日不闷杀、有溢价是判断股性的关键标准。优先选择市场辨识度高、有故事可讲的标的。

  3. 3

    📈 趋势（第三优先级）

    冷门趋势板，有基本面亮点也可以关注。趋势是加分项，但权重低于氛围和股性。当氛围和人气都满足时，趋势好的标的优先选择。

🔧 看盘工具与方法

🛠️

看盘软件：同花顺普通版 + 东方财富极速版（用于看十档盘口和涨停排单）。工具不在多，在于用好。核心是盘感与快速决策能力。

⏰ 打板时机决策流程

  1. 开盘前 — 凌晨或前晚复盘，查看隔夜消息面、外围市场走势
  2. 集合竞价 — 观察竞价强度，判断市场情绪方向
  3. 早盘30分钟 — 最活跃时段，快速扫盘确认今日热点板块
  4. 打板确认 — 不是为打板而打板，必须满足：氛围好 + 个股人气高 + 趋势配合
  5. 盘中调整 — 状态差时主动清仓休息，不强求操作
  6. 尾盘决策 — 氛围不好时尽量等尾盘再做决定

精选问答 — 令胡冲谈交易

#

以下18个问答精选自152个问答对，涵盖选股、打板、复盘、心态、风险等核心维度。

Q如何选择个股？
A

对市场2000多只个股基本面了如指掌，然后每天坚持复盘。个股基本面信息要烂熟于心，大多数上市公司的官方网站我都看过。这样在盘中当某只股票异动时，你能快速判断其基本面是否支持上涨。

Q打板时机如何把握？
A

按重要程度排序：第一看市场氛围，第二看个股人气和股性（近期经常涨停、次日不闷杀），第三看趋势（冷门趋势板有基本面亮点）。不是为打板而打板，而是综合判断后的决策。

Q用什么看盘工具？
A

同花顺普通版看行情，东方财富极速版看十档盘口和涨停排单。关键是盘感，工具是辅助。对盘面的理解比工具本身重要得多。

Q如何复盘？
A

我复盘主要看昨天的涨停板今天的赚钱效应：哪一类涨停股次日有肉，哪一类股持续有肉。不看别人帖子，那些都是别人的观点，你得有自己的观点。复盘核心是判断赚钱效应的持续性。

Q盘感如何培养？
A

长期复盘和市场天天对话的结果。没有什么具体理论可以说明，就是一种潜意识里的感觉，甚至是条件反射。必须每天盯盘、复盘，持续积累。

Q职业交易经历如何？
A

在熊市中练打板了两年多，打过无数烂板，爆亏的次数数不胜数，在无数烂板中学会了打好板。没有捷径，只有实战积累。

Q打板心态如何控制？
A

我打板的时候，没有8成以上把握我是不会出手的。七八成把握的，哪怕涨停了我也不觉得可惜。关键是要有自己的判断标准，不因错过而焦虑。

Q如何记住2000多只股票？
A

"唯手熟尔"，长期形成的记忆烙印。不是刻意去背，而是通过每天复盘、每天看盘、每天研究基本面，自然而然就记住了。大多数上市公司的官网我都看过。

Q大回撤的原因是什么？
A

没认真复盘，懈怠了，总是幻想市场切换。当你不认真复盘的时候，对市场的感知就会下降，该谨慎的时候不谨慎，该出手的时候不出手。懈怠是最致命的亏损根源。

Q你的操作模式是怎么形成的？
A

操作模式源于个性，我不喜欢刻板的框架。模式不是设计出来的，而是长期交易中自然形成的。每个人都有适合自己性格的模式，核心是找到与自己个性匹配的方式。

Q什么是赚钱效应？
A

"瞬息万变中的核心不变才是市场的本质，四个字，赚钱效应。" 昨天涨停的股票今天能否继续赚钱、哪些类型的涨停股持续有肉，这就是赚钱效应。只关注这个，其他消息面的信息都是噪音。

Q资金管理怎么做？
A

我的交易风格是分仓滚动（龙飞虎风格），但偶尔也会集中火力攻击目标股。分仓是常态，集中是特例。核心是在风险和收益之间找到平衡。

Q怎么应对亏损和调整？
A

状态很差的时候，清仓休息。近期有别的事情，基本没看盘，按我这种操作模式，股票上就很难顾得上了。交易不是生活的全部，状态不好的时候强行操作只会更差。

Q复盘看什么？
A

复盘不是看帖子，那些都是别人的观点，你得有自己的观点。主要看：涨停板的赚钱效应、哪类涨停股次日吃肉、连板股的分布。每天都做，形成自己的复盘点。

Q市场氛围差的时候怎么做？
A

跌幅榜上躺着众多跌停或者大跌百分5以上的个股，意味着市场氛围很差。这时候要小心，不轻易出手，尽量等尾盘。宁愿错过，不要做错。

Q职业交易最重要的素质是什么？
A

执行力。知道什么该做什么不该做，然后坚决执行。在熊市中练打板两年多，不断从烂板中学习，最终形成自己的打法。没有超强的执行力，再好的体系也无法落地。

Q怎么看待"错过涨停"的心态？
A

七八成把握的，哪怕涨停了我也不觉得可惜。只有八成以上把握才出手。交易讲究确定性，不追求每一个机会。错过总比做错好，这是交易的第一课。

Q对新手的建议？
A

大多数上市公司的官方网站我都看过。先下苦功夫，把基本盘搞清楚。然后坚持每天复盘，形成自己的判断体系。交易没有捷径，只有持续的学习和积累。

#

氛围确认后的打板操作
市场氛围良好，涨停板赚钱效应延续 → 选择近期有过涨停的股性活跃标的 → 确认趋势配合 → 扫板买入。
核心逻辑：不是为打板而打板，而是市场氛围+个股人气+趋势的三重确认。
盈利卖出 · 龙头股 卖出
赚钱效应衰减时的主动止盈
连板后观察赚钱效应是否持续 → 发现跟风股开始分化 → 核心人气股出现分歧信号 → 主动止盈锁定利润。
关键判断：赚钱效应是决定持有还是卖出的核心指标，不是看股价涨了多少。
空仓休息 · 市场差 空仓
市场氛围恶劣时的空仓
跌幅榜出现众多跌停，市场氛围很差 → 不轻易出手 → 等尾盘观察 → 确认无好转 → 继续空仓。
纪律：即使错过反弹也不可惜，宁愿错过不要做错。
分仓操作 · 滚动 分仓
分仓滚动vs集中攻击
常态是分仓滚动（分散风险，保持灵活性）→ 遇到确定性极高的机会时集中火力攻击目标股。
资金管理原则：分仓是防御，集中是进攻，视市场氛围和个股确定性灵活切换。
回顾反思 · 回撤 反思
两次大回撤的教训
两次较大的回撤，根因共同点：没认真复盘 + 懈怠了 + 总是幻想市场切换。
当复盘质量下降 → 对市场感知变弱 → 该谨慎时不谨慎 → 该果断时不果断 → 回撤。
教训：复盘是交易的燃料，懈怠是亏损的起点。

复盘方法论 — 完整流程

#
💡

令胡冲复盘观："复盘不是看帖子，那些都是别人的观点，你得有自己的观点" — 自主、独立、持续，是复盘的三条底线。

📋 标准化复盘流程（每日执行）

  1. 整理涨停板 — 列出当日所有涨停股，分类标注（连板/首板/烂板/一字板）
  2. 分析赚钱效应 — 昨日涨停股今天的表现：哪类股有肉、哪类股闷杀、溢价率分布
  3. 识别主线 — 哪个板块带动了赚钱效应？是否有持续性热点？
  4. 板块内部分析 — 龙头是谁？跟风是谁？板块梯队是否完整？
  5. 评估市场氛围 — 涨跌家数、涨停/跌停数量、大面股数量、连板高度
  6. 自省交易 — 今天的操作是否符合体系？哪些做得好？哪些可以改进？
  7. 明日预案 — 根据赚钱效应和氛围推演明日可能的方向和应对策略

🔎 复盘的核心观察指标

指标观察重点意义
涨停家数首板 vs 连板比例判断市场活跃度
涨停溢价率昨日涨停股今日开盘/收盘表现赚钱效应的直接指标
连板高度最高连板数及分布情绪周期位置
跌停/大面跌停数量及大跌5%以上个股市场氛围的预警信号
板块效应热点板块的梯队完整性判断行情持续性
成交量整体及板块的成交变化判断资金是否进场

回撤教训 — 两次大回撤的深度反思

#
🚨

大回撤根源：懈怠 + 幻想

"没认真复盘，懈怠了，总是幻想市场切换" — 这是令胡冲对自己两次大回撤的总结。每次回撤前，复盘质量先下降，对市场感知迟钝，进而导致错误的交易决策。

📉 回撤的衍生链条

复盘松懈 → 感知下降 → 幻想切换 → 错误交易 → 大回撤

🔴 回撤教训一

复盘不可断：每天必须复盘，连续复盘是保持市场敏感度的唯一方式。一日不复盘，三日白交易。

🔴 回撤教训二

警惕"幻想切换"：当市场风格不适合自己模式时，不要幻想马上切换，而是主动降低仓位、减少操作，等待环境明朗。

🔴 回撤教训三

状态不佳即停："状态很差，清仓休息" — 交易状态与结果直接相关，状态差时强行操作只会加速亏损。

🔴 回撤教训四

敬畏市场氛围：氛围差时不出手，这是铁律。再好的个股在市场暴跌面前都是脆弱的。顺势而为，不逆势硬来。

令胡冲交易检查清单

#

每次出手前，逐条核对。全部通过才考虑交易，任何一条不满足→暂缓或放弃。

☀️ 市场氛围是否良好？

跌幅榜有无众多跌停或大跌5%以上的个股？大盘走势是否健康？涨跌家数比是否合理？

💰 赚钱效应是否持续？

昨日涨停股今天表现如何？溢价率是否为正？连板股是否如期晋级？

🔥 目标个股人气是否足够？

近期有没有涨停记录？股性是否活跃？次日是否不闷杀？市场辨识度如何？

📈 趋势配合吗？

个股处于上涨趋势中吗？如果是冷门股，是否有基本面亮点支撑？

🎯 是否达到八成以上把握？

综合来看，这笔交易有80%以上的把握吗？犹豫就不做，错过不可惜。

🧠 今日复盘是否完成？

今天有没有认真复盘？对市场的感知是否清晰？有没有因为懈怠而降低标准？

💪 状态是否在线？

精神状态是否良好？有没有情绪干扰？如果状态很差，有没有清仓休息的勇气？

📊 资金管理是否合理？

这次出手的仓位是否在合理范围内？当前是分仓滚动还是集中攻击？是否匹配当前市场环境？

❌ 不满足1项 → 放弃 ❌ 不满足2-3项 → 暂缓 ✅ 全部满足 → 执行

记住令胡冲的核心纪律：没有8成以上把握绝不出手，哪怕涨停也不觉得可惜。
